Segmentation Parameters

Name Description
Level-set iterations The default value for this parameter is 200, which works for most cases. If the image quality is poor and over-segmentation is observed, please reduce this number. Otherwise, increase the value.
Contour length The shortest length of cell contours. If a cell contour length is less than this value, then the result is discarded.
Field of view Field of view that the image was acquired with (typically 0.5 to 3.0 deg.) For example, a 1.0 means that an image was acquired with a 1.0 deg. field of view with 750⨯605 pixels. This parameter should be scaled if the pixel sampling differs.

Keyboard Shortcuts and Mouse Operations

Key Action Description
F1  Keyboard Shortcuts... Display this help screen.
F2 Toggle Visibility Temporarily switch off/on visibility of segmented cones. Visibility is automatically restored when you open or segment images, as well as open the Settings dialog.
F7  Snapshot... Open a preview dialog displaying current image with annotations. You can select the desired scale using a slider at the bottom, then click the “Save” button. You will be asked for a .png file name to save the snapshot to.
F10 Reset Image View In the “Default” mouse mode dragging the mouse changes brighness/contrast of the image. Pressing F10 reverts these changes and resets the image display to default settings. Also helpful if you zoom in/out too much or accidentally rotate the image (with Ctrl+left-dragging), and you want the origional view back.
Ctrl+I Toggle Interpolation Togle pixel interpolation for image zoom. If the interpolation is on, a bicubic resampling is used. Otherwise, nearest neighbor.
Ctrl+O  Open Open Split Image (.tif) and, optionally, Annotations (_contours.csv). Displays a dialog where you can select source image directory, annotations directory, and check images that you want to open. Anntotations files are automatically matched with the image files on file name without “.tif” or “_contours.csv”.
Ctrl+L Show data file locations Displays the “Image Data Location” box showing full paths to the currently displayed image, its associated contour spreadsheet in the same directory, and auto-backup contour spreadsheet. The auto-backup (history) file is updated after each contour operation, and it is loaded when there is no manually saved spreadsheet in the image file directory. It prevents your editings from being lost when you accidentally close the program (or open new images) without saving the data.
Ctrl+G  Segment Display Cone Segmentation dialog. On this dialog you can adjust segmentation parameters, and choose one or more open images to segment cones on. Note that clicking on the little checkmark √ in the table header checks/unchecks all images in the list.
Ctrl+S  Save Save segmentation results of all open images.
Ctrl+M  Default Switch to “Default” mouse mode. In this mode:
– Mouse wheel (or Ctrl+right-dragging) zooms in and out;
– Dragging the mouse changes brightness/contrast;
– Shift+left-dragging (or middle-dragging) scrolls the image.
– Ctrl+left-dragging (Windows) rotates the image.
Ctrl+C  Draw Switch to “Draw Cone Contours” mouse mode. In this mode:
– Mouse wheel (or Ctrl+right-dragging) zooms in and out;
– Dragging the mouse around a cone adds a new contour;
– Shift+left-dragging (or middle-dragging) scrolls the image.
– Ctrl+left click (Windows) or Alt+left click (Mac OS X) on a contour erases the contour.
Ctrl+E  Edit Switch to “Edit Cone Contours” mouse mode. In this mode:
– Mouse wheel (or Ctrl+right-dragging) zooms in and out;
– Left click on a contour selects it for editing;
– When editing, you can edit the edges by left-dragging them;
– When editing, you can move the entire contour by middle-dragging it by an edge;
– Ctrl+left click (Windows) or Alt+left click (Mac OS X) on a contour erases the contour.
– Shift+left-dragging (or middle-dragging) scrolls the image.
Ctrl+D  Erase M Switch to “Erase Multiple Cone Contours” mouse mode. In this mode:
– Mouse wheel (or Ctrl+right dragging) zooms in and out;
– Left-dragging the mouse around an area erases all contours inside this area;
– Shift+left-dragging (or middle-dragging) scrolls the image.
Ctrl+W  Erase S Switch to “Erase Single Cone Contour” mouse mode. In this mode:
– Mouse wheel (or Ctrl+right-dragging) zooms in and out;
– Left-clicking on a contour erases it;
– Shift+left-dragging (or middle-dragging) scrolls the image.
Ctrl+Z  Undo Undo the last “Draw” or “Erase” operation.
